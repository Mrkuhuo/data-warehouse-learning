# 开发者工具集合

一个简单实用的开发者工具集合网站，参考了json.cn的设计风格，提供了多种常用的开发工具。

## 功能列表

1. **代码格式化及压缩工具**
   - 支持JSON、SQL、JavaScript、XML格式
   - 自动语法高亮显示
   - 一键格式化和压缩
   - 支持自动格式化

2. **日期时间戳转换工具**
   - 获取当前时间和时间戳
   - 日期转时间戳（毫秒级）
   - 时间戳转日期

3. **代码对比工具**
   - 直观展示两段代码的差异
   - 差异部分高亮显示

4. **URL编解码工具**
   - URL编码
   - URL解码

5. **代码转换工具**
   - JSON转Java
   - SQL转Java

## 使用方法

直接打开 `index.html` 文件即可使用所有功能，无需安装任何依赖。

## 技术说明

- 纯原生JavaScript实现，无任何框架依赖
- 响应式设计，适配不同设备
- 本地运行，数据不会上传到服务器

## 浏览器兼容性

推荐使用Chrome浏览器获得最佳体验。
